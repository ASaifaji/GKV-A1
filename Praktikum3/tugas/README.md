### How to Use,

```
w = arm extension
s = arm flexion
e = forearm extension
d = forearm flexion
r = wrist extension
f = wrist flexion
t = thumb extension
g = thumb flexion
y = index finger extension
h = index finger flexion
u = middle finger extension
j = middle finger flexion
i = ring finger extension
k = ring finger flexion
o = pinky finger extension
l = pinky finger flexion
esc = exit
```
